# BRF Bonneville

## Development

We recommend using [pnpm](https://pnpm.io/) to save disk space on your computer.

### 1. Install Dependencies

```bash
pnpm install
```

### 2. Start development Server

```bash
pnpm dev
```

### 3. Preview & Build

```bash
pnpm build
pnpm preview
```

### Other Commands

```bash
pnpm astro add
pnpm astro --help
```
